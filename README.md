# Etch-A-Sketch

A simple Etch-A-Sketch board with color picker, random color, eraser , clear board and adjustable grid size buttons.

![etch-a-sketch](./img/screenshot.png)

[Live Preview](https://vsilagy.github.io/etch-a-sketch)

- CSS
- JavaScript
